# 🎙️ AI Voice Journalist

AI-powered app that generates **audio news briefings** from real-time News and Reddit topics using LLMs and Text-to-Speech.

---

## 🚀 Features

- 📰 Fetch real-time News data  
- 💬 Get Reddit discussions  
- 🤖 AI summarization using LLMs (Groq)  
- 🎧 Convert summaries into audio (ElevenLabs)  
- 🧠 Combine multiple topics into one briefing  
- 🖥️ Simple Streamlit interface  

---

## 🏗️ Tech Stack

- Streamlit (Frontend)  
- FastAPI (Backend)  
- Groq / LLM APIs (Summarization)  
- Exa API (Search)  
- ElevenLabs (Text-to-Speech)  
- Python
